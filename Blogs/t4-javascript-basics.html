HOW DOES JAVASCRIPT COMPARE TO HTML AND CSS?

HTML, is used to create the basic structure and content of a webpage. Like the framing does in your house.

CSS, is used for the design of a webpage – where everything is placed and how it looks. So like in your house, it would mean which room is which, and what colour the walls are.

JAVASCRIPT, creates interactivity! So if we take your house it would mean that when you pushed the garage door remote button the door would go up or down.

Loops

Loops. What is a loop? JavaScript loops are used to repeatedly run a block of code - until a certain condition is met. So like driving around the block time and again until a park becomes available.

WHAT IS CONTROL FLOW?

control flow is the order in which the computer executes statements in a script. It's like a task list at work or a check list, which sets out the order in which things must be done, or in order to brush your teeth you must first put tooth paste on the tooth brush.

THE DIFFERENCE BETWEEN ACCESSING DATA FROM ARRAYS AND OBJECTS.

Arrays are a special type of objects, with numbered indexes. Arrays use numbered indexes while Objects use named indexes.

You should use objects when you want the element names to be strings (text), and You should use arrays when you want the element names to be numbers.

You access an array element by referring to the index number.

WHAT ARE FUNCTIONS AND WHY ARE THEY USEFUL?

Functions are one of the fundamental building blocks in JavaScript. A function is a JavaScript procedure—a set of statements that performs a task or calculates a value. To use a function, you must define it somewhere in the scope from which you wish to call it.
